~~~git新学命令~~~

git commit --amend --no-edit :提交 追加到上次变更文件不改备注(需要确认是否产生新版本)


git commit -m "配置本地热部署，添加相关依赖。"


[master a49acd9] 配置本地热部署，添加相关依赖。


执行：git commit --amend --no-edit


[master a251d23] 配置本地热部署，添加相关依赖。


可以看到这个命令：仅仅是作为本地连续commit时的不需要重新备注的操作。当然这个细节还是挺好的
